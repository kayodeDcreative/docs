## Setup Instructions

### 1. Install Gately Library

Add the Gately component library to your Framer project:

```javascript
// In your Framer project
// Add Gately components from our library
// or import from: https://framer.com/gately
```

### 2. Configure Project Settings

Set up your Gately project configuration:

```javascript
// Project Configuration
const gatelyConfig = {
  projectId: "your-project-id",
  apiUrl: "https://api.gately.com",
  redirectUrl: "https://your-domain.com",
  // Optional: Custom styling
  theme: {
    primaryColor: "#6b7280",
    borderRadius: "8px",
    fontFamily: "Onest, sans-serif"
  }
}
```

### 3. Add Authentication Flow

Create a complete authentication flow:

<CodeGroup>

```javascript Sign Up Form
<GatelySignUpForm 
  projectId="your-project-id"
  onSuccess={(user) => {
    // Handle successful signup
    console.log("User signed up:", user)
  }}
/>
```

```javascript Sign In Form
<GatelySignInForm
  projectId="your-project-id"
  onSuccess={(user) => {
    // Handle successful signin
    console.log("User signed in:", user)
  }}
/>
```

```javascript User Profile
<GatelyUserProfile
  projectId="your-project-id"
  onUpdate={(user) => {
    // Handle profile updates
    console.log("Profile updated:", user)
  }}
/>
```

</CodeGroup>

## Component Examples

### Simple Sign Up Form

A clean, minimal sign up form with email and password fields:

```javascript
<GatelySignUpForm
  projectId="your-project-id"
  fields={["email", "password", "name"]}
  theme={{
    primaryColor: "#6b7280",
    borderRadius: "12px",
    fontFamily: "Onest, sans-serif"
  }}
  onSuccess={(user) => {
    // Redirect to dashboard
    window.location.href = "/dashboard"
  }}
  onError={(error) => {
    // Show error message
    console.error("Sign up error:", error)
  }}
/>
```

### User Profile with Plans

A comprehensive user profile with plan management:

```javascript
<GatelyUserProfile
  projectId="your-project-id"
  showPlans={true}
  showBilling={true}
  theme={{
    primaryColor: "#6b7280",
    cardStyle: "elevated",
    spacing: "comfortable"
  }}
  onPlanChange={(plan) => {
    // Handle plan changes
    console.log("Plan changed to:", plan)
  }}
  onBillingUpdate={(billing) => {
    // Handle billing updates
    console.log("Billing updated:", billing)
  }}
/>
```

## Styling Options

Customize the appearance of Gately components to match your brand:

```javascript
const theme = {
  // Colors
  primaryColor: "#6b7280",
  secondaryColor: "#f3f4f6",
  textColor: "#111827",
  backgroundColor: "#ffffff",
  
  // Typography
  fontFamily: "Onest, sans-serif",
  fontSize: "16px",
  fontWeight: "400",
  
  // Layout
  borderRadius: "8px",
  spacing: "16px",
  cardStyle: "elevated", // "flat", "elevated", "outlined"
  
  // Form elements
  inputStyle: "outlined", // "filled", "outlined"
  buttonStyle: "solid", // "solid", "outlined", "ghost"
  
  // Responsive
  breakpoints: {
    mobile: "768px",
    tablet: "1024px"
  }
}
```

## Advanced Configuration

### Custom Authentication Flow

Create a multi-step authentication process:

```javascript
<GatelyAuthFlow
  projectId="your-project-id"
  steps={[
    {
      type: "signup",
      fields: ["email", "password", "name"],
      validation: {
        email: "required|email",
        password: "required|min:8",
        name: "required|min:2"
      }
    },
    {
      type: "verification",
      method: "email"
    },
    {
      type: "profile",
      fields: ["company", "role", "phone"],
      optional: true
    }
  ]}
  onComplete={(user) => {
    console.log("Authentication flow completed:", user)
  }}
/>
```

### SSO Integration

Add social login options:

```javascript
<GatelySSOButtons
  projectId="your-project-id"
  providers={["google", "github", "microsoft"]}
  theme={{
    layout: "horizontal", // "vertical", "horizontal", "grid"
    buttonStyle: "outlined",
    showLabels: true
  }}
  onSuccess={(user, provider) => {
    console.log(`Logged in with ${provider}:`, user)
  }}
/>
```


## Troubleshooting

  <Accordion title="Components not loading">

    - Verify internet connection
