# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added

- **Helpdesk Email Integration**: Customers can reply to tickets via email. ([Read more](/helpdesk-inbound-email-v1))
- **Helpdesk Agent Email Replies**: Agents can reply to tickets via email. ([Read more](/helpdesk-inbound-email-v1-1))
- **AI Agent System**: Autonomous AI agent for smart task execution.
- **Rich Text Editor Enhancements**: Table support, advanced callouts, and more.
- **E-commerce Shop System**: Complete product and order management.
- **Email Campaign System**: Campaign creation with a visual editor.
- **Form Builder System**: Drag-and-drop form builder with custom fields.
- **Gated Content System**: Content access control based on plans or authentication.
- **Community & Discussion System**: Channel-based discussions and comment threads.
- **Webhook System**: Event-based triggers and payload tracking.
- **Integration Hub**: Slack, Mailchimp, Circle, and Stripe integrations.
- **Redirect Rules System**: URL redirect management.
- **Mobile-Optimized Interface**: Responsive mobile layouts for all major features.
- **AI Article Drafts Feature**: AI-powered article generation from helpdesk tickets.
- **Customer Feedback System**: Public feedback submission and management dashboard.
- **Help Article Categories Enhancement**: Icon support and category descriptions.
- **Help Center Settings - Header & Hero Configuration**: Customize the header and hero sections.
- **Widget Icon System Migration**: Migrated from Lucide to Phosphor Icons.

### Changed

- **Widget Installation Simplified**: Streamlined UI for widget setup.
- **API Improvements**: PATCH method support for widgets API and other enhancements.
- **UI/UX Enhancements**: Skeleton loaders, better loading states, and improved form validation.

### Fixed

- **Widgets API 405 Error**: Updated API to accept both PUT and PATCH methods.
- **Help Articles Fetching**: Fixed issue with articles not loading.
- **Image Upload vs URL Input**: Implemented upload components for all image fields.
