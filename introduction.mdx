## For No-Code and Developers

Gately is designed for everyone, whether you're a no-code creator or a developer building a custom solution.

### No-Code Creators
- **Seamless Integration:** Works with Framer, Webflow, Bubble, and more.
- **Visual Setup:** Configure everything through our intuitive dashboard.
- **Pre-built Components:** Use our ready-made components to get started in seconds.

### Developers
- **Powerful SDK:** Use our JavaScript SDK for deep, custom integrations.
- **Full API Access:** Leverage our REST API to build any workflow you can imagine.
- **Framework Support:** Compatible with React, Vue, Svelte, and vanilla JS.
