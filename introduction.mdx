# Introduction

The Gately SDK is a universal authentication and user management solution that works across all major frameworks and platforms. It provides a simple, consistent API for handling user authentication, form submissions, analytics, and more.

## What is Gately?

Gately is a complete authentication platform that provides:

- **Universal Authentication** - Works in browsers and Node.js
- **Framework Agnostic** - React, Vue, Svelte, Angular, and vanilla JS support
- **SSO Integration** - Google, GitHub, and custom OAuth providers
- **Form Submissions** - Collect and manage form data
- **Analytics** - Track user behavior and conversions
- **Paywall & Subscriptions** - Manage access control and billing
- **Auto Session Management** - Automatic token refresh and persistence

## Quick Start

### 1. Add the SDK

Choose one of these methods:

```html
<!-- Script Tag (Recommended) -->
<script src="https://cdn.usegately.com/gately.min.js"></script>
```

```javascript
// ES Module Import
import { GatelyBrowserClient } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'
```

### 2. Initialize

```javascript
const gately = new GatelyBrowserClient('your-project-id')
```

### 3. Authenticate

```javascript
// Login with email/password
const { user, session } = await gately.login('user@example.com', 'password')

// Check authentication status
if (gately.isAuthenticated()) {
  console.log('User is logged in:', gately.getUser())
}
```

## Framework Support

The Gately SDK works with all major frameworks:

### React
```jsx
import { useGately } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

function App() {
  const { user, login, logout, isAuthenticated } = useGately('your-project-id')
  // Your component logic
}
```

### Vue
```javascript
import { useGately } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

export default {
  setup() {
    const { user, login, logout, isAuthenticated } = useGately('your-project-id')
    return { user, login, logout, isAuthenticated }
  }
}
```

### Svelte
```javascript
import { createGatelyStore } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

const gatelyStore = createGatelyStore('your-project-id')
```

### Angular
```typescript
import { GatelyService } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

@Injectable({
  providedIn: 'root'
})
export class GatelyService {
  // Service implementation
}
```

Full TypeScript support is included:

```typescript
import { GatelyBrowserClient, User, Session } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

const gately = new GatelyBrowserClient('your-project-id')

async function handleLogin(): Promise<{ user: User; session: Session }> {
  return await gately.login('user@example.com', 'password')
}
```
