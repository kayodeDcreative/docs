## What is Gately?

Gately is the easiest way to add authentication and user management to your Framer sites. Built specifically for Framer users, Gately eliminates the complexity of user authentication while providing enterprise-grade security and features.

### Built for Framer

Gately was designed from the ground up specifically for Framer users. Unlike complex authentication services that require coding knowledge, Gately works with just one simple script added to your Framer site.

- **One-script installation** - Add to your site header and you're done
- **Automatic integration** - Works seamlessly with your Framer design
- **Visual customization** - Configure everything through our dashboard
- **Instant activation** - Publish your site and authentication is live

### No Coding Required

Perfect for designers, entrepreneurs, and anyone building with Framer who needs professional authentication without the technical complexity.

- **Dashboard configuration** - Set up everything visually
- **Automatic styling** - Matches your site's design automatically
- **Enterprise security** - Bank-level security without the complexity
- **User management** - Complete user profiles, plans, and billing

## Quick Start

Add authentication to your Framer site in under 2 minutes:

<CodeGroup>

```html Framer Installation
<!-- Get your personalized script from the dashboard -->
<script 
  src="https://cdn.usegately.com/gately.min.js" 
  data-project-id="your-project-id"
  data-domain="https://yoursite.framer.website"
  data-base-domain="yourdomain.com"
  data-slug="your-slug">
</script>
```


```javascript Advanced Usage (Optional)
// For developers who need programmatic access
npm install @gately/sdk

import { GatelyBrowserClient } from '@gately/sdk'

const gately = new GatelyBrowserClient('your-project-id')

// Check authentication status
if (gately.isAuthenticated()) {
  console.log('User is logged in:', gately.getUser())
}
```

</CodeGroup>

### Installation Steps

1. **Get your script** from [app.usegately.com/dashboard/settings/installation](https://app.usegately.com/dashboard/settings/installation)
2. **Add to Framer** - Go to Settings → Custom Code → paste in `<head>` section
3. **Publish your site** - Authentication is now live\!
