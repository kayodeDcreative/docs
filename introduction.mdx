# Introduction

Gately is a complete authentication and user management platform designed for modern websites and no-code applications. Whether you're building with Framer, Webflow, Bubble, or any other platform, Gately provides seamless user authentication, form handling, and access control without requiring any coding.

## How It Works

### 1. Add Gately to Your Site
Simply add a small script to your website's header. No coding required.

### 2. Configure Your Settings
Use the Gately dashboard to customize authentication, forms, and user experience.

### 3. Users Can Sign Up and Login
Gately automatically handles user registration, login, and session management.

### 4. Manage Everything from Dashboard
View users, form submissions, analytics, and manage subscriptions all in one place.
