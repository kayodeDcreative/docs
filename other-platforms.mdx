## Platform Categories

### Website Builders
These platforms support custom code injection:

- **Wix** - Add custom HTML/JavaScript
- **Squarespace** - Custom code blocks
- **Shopify** - Theme customization
- **WordPress** - Plugin or theme integration
- **Carrd** - Custom code sections
- **Notion** - Embed custom websites

### No-Code Platforms
These platforms support API integration:

- **Airtable** - API integration for user data
- **Zapier** - Connect Gately to 5000+ apps
- **Make (Integromat)** - Visual workflow automation
- **n8n** - Open-source workflow automation
- **Retool** - Build internal tools with Gately data

### Custom Websites
For custom-built websites:

- **Static Sites** - Add Gately script to HTML
- **React/Vue/Angular** - Use Gately SDK
- **PHP/Node.js** - Server-side integration
- **Mobile Apps** - API integration

## Integration Methods

### Method 1: Script Tag (Recommended)

For platforms that support custom HTML/JavaScript:

1. **Get your embed script** from the Gately dashboard
2. **Add to your platform's custom code section**
3. **Publish your site** - Gately will be active!

### Method 2: API Integration

For platforms that support API connections:

1. **Get your API credentials** from the Gately dashboard
2. **Configure API connector** in your platform
3. **Set up authentication endpoints**
4. **Test and deploy**

### Method 3: Custom Implementation

For advanced users:

1. **Use Gately's JavaScript SDK**
2. **Build custom authentication flows**
3. **Integrate with your existing codebase**
4. **Deploy and test**

## Platform-Specific Guides

### Wix Integration

1. Go to **Settings** → **Custom Code**
2. Add your Gately script to the **`<head>` section**
3. Publish your site

### Squarespace Integration

1. Go to **Settings** → **Advanced** → **Code Injection**
2. Add your Gately script to the **Header**
3. Save and publish

### Shopify Integration

1. Go to **Online Store** → **Themes** → **Actions** → **Edit Code**
2. Add your Gately script to `theme.liquid` in the `<head>` section
3. Save and publish

### WordPress Integration

1. Install a custom code plugin (like "Code Snippets")
2. Add your Gately script to the header
3. Activate the snippet

## Testing Your Integration

After adding Gately to your platform:

1. **Test on your platform** - Verify everything works in your environment
2. **Test authentication flows** - Create test accounts
3. **Check the dashboard** - Verify users appear in your Gately dashboard
4. **Test on mobile** - Ensure everything works on different devices
5. **Test on different browsers** - Verify cross-browser compatibility

## Troubleshooting

<AccordionGroup>
  <Accordion title="Script not loading">
    - Check your platform's custom code settings
    - Verify the script is in the correct location
    - Check for JavaScript errors in browser console
    - Test with a simple script first
  </Accordion>
  
  <Accordion title="Styling conflicts">
    - Check for CSS conflicts with your platform
    - Test with default Gately styling first
    - Adjust custom CSS if needed
    - Contact support for platform-specific issues
  </Accordion>
  
  <Accordion title="Authentication not working">
    - Verify domain is whitelisted in dashboard
    - Check browser console for errors
    - Test with different browsers
    - Ensure platform allows external scripts
  </Accordion>
  
  <Accordion title="Mobile responsiveness issues">
    - Test on actual devices
    - Check responsive breakpoints
    - Verify touch interactions work
    - Test platform's mobile preview
  </Accordion>
</AccordionGroup>
