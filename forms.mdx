# Forms

Gately provides a complete form management system that allows you to create, customize, and handle form submissions with built-in analytics and user tracking.

## Overview

The forms system includes:
- **Form Builder**: Create custom forms with various field types
- **Form Submissions**: Handle and process form data
- **Analytics**: Track form performance and user interactions
- **Notifications**: Get notified of new submissions
- **Customization**: Style forms to match your design

## Creating Forms

Forms can be created through the Gately dashboard or programmatically via the API.

### Dashboard Form Builder

1. Go to your Gately dashboard
2. Navigate to **Forms** section
3. Click **Create New Form**
4. Add fields and configure settings
5. Copy the form ID for SDK integration

## SDK Integration

### Submit Form Data

```javascript
import { GatelyBrowserClient } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

const gately = new GatelyBrowserClient('your-project-id')

// Submit form data
async function handleFormSubmit(formData) {
  try {
    const response = await gately.submitForm('your-form-id', {
      name: formData.name,
      email: formData.email,
      message: formData.message,
      // Add any other form fields
    })
    
    console.log('Form submitted successfully:', response)
    
    // Handle success (redirect, show message, etc.)
    if (response.redirect_url) {
      window.location.href = response.redirect_url
    }
  } catch (error) {
    console.error('Form submission failed:', error)
  }
}
```

### Form with Validation

```html
<form id="contact-form" onsubmit="handleSubmit(event)">
  <input 
    type="text" 
    name="name" 
    placeholder="Your Name" 
    required
  />
  <input 
    type="email" 
    name="email" 
    placeholder="Your Email" 
    required
  />
  <textarea 
    name="message" 
    placeholder="Your Message" 
    required
  ></textarea>
  <button type="submit">Send Message</button>
</form>

<script>
async function handleSubmit(event) {
  event.preventDefault()
  
  const form = event.target
  const formData = new FormData(form)
  
  // Convert FormData to object
  const data = Object.fromEntries(formData.entries())
  
  try {
    const response = await gately.submitForm('contact-form-id', data)
    
    // Show success message
    alert(response.message || 'Thank you for your submission!')
    form.reset()
  } catch (error) {
    alert('Submission failed: ' + error.message)
  }
}
</script>
```

## React Integration

```jsx
import React, { useState } from 'react'
import { useGately } from 'https://cdn.usegately.com/gately-sdk.esm.min.js'

function ContactForm() {
  const { submitForm } = useGately('your-project-id')
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    message: ''
  })
  const [loading, setLoading] = useState(false)
  const [message, setMessage] = useState('')

  const handleSubmit = async (e) => {
    e.preventDefault()
    setLoading(true)
    setMessage('')

    try {
      const response = await submitForm('contact-form-id', formData)
      setMessage(response.message || 'Thank you for your submission!')
      setFormData({ name: '', email: '', message: '' })
    } catch (error) {
      setMessage('Submission failed: ' + error.message)
    } finally {
      setLoading(false)
    }
  }

  const handleChange = (e) => {
    setFormData({
      ...formData,
      [e.target.name]: e.target.value
    })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        name="name"
        placeholder="Your Name"
        value={formData.name}
        onChange={handleChange}
        required
      />
      <input
        type="email"
        name="email"
        placeholder="Your Email"
        value={formData.email}
        onChange={handleChange}
        required
      />
      <textarea
        name="message"
        placeholder="Your Message"
        value={formData.message}
        onChange={handleChange}
        required
      />
      <button type="submit" disabled={loading}>
        {loading ? 'Sending...' : 'Send Message'}
      </button>
      {message && <p>{message}</p>}
    </form>
  )
}
```

## Form Analytics

Gately automatically tracks form submissions and provides analytics:

```javascript
// Form submission is automatically tracked
await gately.submitForm('form-id', data)

// You can also manually track form interactions
gately.trackEvent('form_view', {
  form_id: 'contact-form',
  page_url: window.location.href
})

gately.trackEvent('form_start', {
  form_id: 'contact-form'
})
```

## Form Settings

### Success Message

Configure custom success messages in your form settings:

```javascript
// Form will return this message on successful submission
const response = await gately.submitForm('form-id', data)
console.log(response.message) // "Thank you for contacting us!"
```

### Redirect After Submission

Set up automatic redirects after form submission:

```javascript
// Form settings can include a redirect URL
const response = await gately.submitForm('form-id', data)

if (response.redirect_url) {
  window.location.href = response.redirect_url
}
```

### Email Notifications

Enable email notifications for new form submissions in your dashboard settings.

## Advanced Usage

### Custom Form Validation

```javascript
async function validateAndSubmit(formData) {
  // Custom validation
  if (formData.email && !formData.email.includes('@')) {
    throw new Error('Invalid email address')
  }
  
  if (formData.message && formData.message.length < 10) {
    throw new Error('Message must be at least 10 characters')
  }
  
  // Submit if validation passes
  return await gately.submitForm('form-id', formData)
}
```

### File Uploads

```javascript
// Handle file uploads
async function handleFileUpload(file) {
  // Upload file to your storage service
  const uploadedUrl = await uploadToStorage(file)
  
  // Submit form with file URL
  await gately.submitForm('form-id', {
    ...formData,
    file_url: uploadedUrl
  })
}
```

### Conditional Form Fields

```javascript
// Show/hide fields based on user input
function toggleFieldVisibility() {
  const fieldType = document.getElementById('field-type').value
  const customField = document.getElementById('custom-field')
  
  if (fieldType === 'custom') {
    customField.style.display = 'block'
  } else {
    customField.style.display = 'none'
  }
}
```

## Error Handling

```javascript
try {
  const response = await gately.submitForm('form-id', data)
  // Handle success
} catch (error) {
  switch (error.code) {
    case 'FORM_NOT_FOUND':
      console.error('Form does not exist')
      break
    case 'VALIDATION_ERROR':
      console.error('Form validation failed:', error.details)
      break
    case 'RATE_LIMITED':
      console.error('Too many submissions, please try again later')
      break
    default:
      console.error('Form submission failed:', error.message)
  }
}
```
